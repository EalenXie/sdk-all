SDK ALL
=======
此项目为个人研发并总结所有对接第三方请求接口SDK封装整合项目

> 💾 设计目的和场景
>
> - 节省接口对接成本,按需引入即用
> - 让使用者更加专注于业务开发,底层SDK与业务代码解耦
> - 核心主要依赖`Spring-web`原生特性,`Spring`项目
> - 项目可持续迭代发展

> - 🔁 项目同步维护：[Github](https://github.com/EalenXie/sdk-all)
> - 📖 电子书阅读：[Github Pages](https://dunwu.github.io/db-tutorial/)
